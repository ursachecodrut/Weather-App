# Weather App With Redux

## Technologies used:

- Flutter
- Redux:
    - Built collection
    - Freezed
    - Epics

## APIs used:

- [Geolocation API](https://ip-api.com/docs/api:json)
- [Open Weather](https://openweathermap.org/api/one-call-api)
